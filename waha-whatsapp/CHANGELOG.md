# Changelog

## 1.0.0

- Initial release
- Send text messages via WhatsApp
- Send images with captions
- Send videos with captions
- NOWEB engine support (no Chromium needed)
- Session persistence across restarts
- Configurable webhook for incoming messages
- Support for amd64 (Synology NAS, Intel NUC) and aarch64 (Raspberry Pi 4)
